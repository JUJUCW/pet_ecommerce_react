# Pet E-commerce React Website

一個簡易的寵物領養網站，包含待領養的狗夠資料、寵物週邊商品。

## Description

* 多種狗狗品種：網站擁有多種狗狗品種，無論你喜歡小型犬、大型犬還是中型犬，你都能在這裡找到適合的夥伴。

* 個性化搜尋：使用搜尋功能，輸入狗狗名稱或關鍵詞，快速找到心儀的狗狗。只需幾秒鐘，就能看到符合要求的所有寵物。

* 詳細信息：每個狗狗都有詳細的資料頁面，包括年齡、大小、顏色、性別等等。可以充分了解每隻狗狗的背景和特點，從而更好地做出決定。

* 點擊跳轉：當找到心儀的狗狗後，只需點擊狗狗的名稱或圖片，將跳轉到詳細的狗狗資訊頁面。

## Getting Started

### Executing program

```
git clone https://github.com/JUJUCW/pet_ecommerce_react.git
```
### Enter the project folder
```
cd pet_ecommerce_react
```

### Installing
```
npm install
```
### Start the project
```
npm start
```
### Start
```
在瀏覽器中打開 http://localhost:3000/pet_ecommerce_react
```
如無自動開啟可由網頁輸入以上網址
### Exit
```
control + C
```
